# 海大拍賣網站

前端網址: (https://davidloman.jamessu2016.com/))

---

## 專案簡介
這是一個可以提供使用者上傳、拍賣物品、競拍、購物的線上拍賣網站。  
前端使用 React(typescript)，後端框架使用 Spring Boot 並連結 MongoDB 資料庫，部署在 cloudflare tunnel 上。  
並使用JWT token驗證使用者身分

---

## 主要功能

### 使用者功能
- 註冊 / 登入 / 登出，使用 JWT 認證  
- 查看所有正在拍賣的商品  
- 參與競標  
- 加入 / 更新購物車  
- 與賣家溝通並下單
- 購賣商品(可套用優惠券)
- 輪盤抽取優惠券
- 查看歷史紀錄:包含瀏覽、購物、競標、評論、訂單紀錄
- 商品收藏紀錄

### 商品管理
- 上傳商品  
- 設定拍賣模式、起標價與結束時間  
- 自動更新拍賣狀態  

### 拍賣與競標
- 即時更新目前最高出價  
- 拍賣結束後自動建立訂單並通知得標者與賣家  

---

## 特色功能
- RAG搜尋:在搜尋商品使用RAG(擷取增強生成)，提供更精確、低幻覺率的搜尋  
- 優惠券：根據使用者消費狀況、特定節日等不定期給予優惠券，優惠券由一個輪盤抽出來，輪盤內不見得都是好的優惠券
- 買賣雙方聊天介面:用websocket技術實現買賣家即時溝通，同時資料庫保存使用者的對話紀錄，離線後重新登入仍可瀏覽聊天歷史
- 託管圖片CDN:用jsdelivr託管圖片，提高網站效能與增加安全性


## 開發提示
### 更新後端swagger方法
1.後端api更新後再port:8080上運行
2.修改orval.config.ts中的input為新版swagger.json的網址
3.前端根目錄下執行npx orval

### 使用playWright測試
1.執行指令下載playwright
npx playwright install
2.執行指令跑測試腳本
npx playwright test --ui
