# 海大拍賣網站

前端網址: [https://se-project2025-frontend-8qpw.vercel.app/](https://se-project2025-frontend-8qpw.vercel.app/)

---

## 專案簡介
這是一個可以提供使用者上傳、拍賣物品、競拍的線上網站。  
前端使用 React，後端框架使用 Spring Boot 並連結 MongoDB 資料庫，部署在 Vercel 上。  
目前已改良主頁的 index 與 header，使主頁能連到購物車，並新增了購物車頁面（尚未實作功能）。

---

## 主要功能

### 使用者功能
- 註冊 / 登入 / 登出，使用 JWT 認證  
- 查看所有正在拍賣的商品  
- 參與競標  
- 加入 / 更新購物車  
- 與賣家溝通並下單  

### 商品管理
- 上傳商品  
- 設定拍賣模式、起標價與結束時間  
- 自動更新拍賣狀態  

### 拍賣與競標
- 即時更新目前最高出價  
- 拍賣結束後自動建立訂單並通知得標者與賣家  

---

## 特色功能
- 商品推薦：根據使用者最近的瀏覽紀錄，推薦使用者可能感興趣的商品  
- 優惠券：根據使用者消費狀況、生日等不定期給予優惠券，優惠券由一個輪盤抽出來，輪盤內不見得都是好的優惠券
